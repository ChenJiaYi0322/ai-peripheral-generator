# AI Peripheral Showcase Generator

This project is a simple AI-assisted tool for generating product showcase content for computer peripherals.

## Features
- Generate product descriptions using AI-style prompts
- Clean Apple-style UI
- Fast frontend prototype

## Tech Stack
- HTML / Tailwind CSS / JavaScript

## Purpose
This project is part of my exploration into AI-assisted development workflows, including:
- Prompt-based UI generation
- Rapid prototyping
- AI content generation

## Future Plans
- Integrate real LLM APIs (OpenAI / DeepSeek / MiMo)
- Build automated workflows (AI agents)
- Expand into a full product showcase platform
